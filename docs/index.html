    Installation

    1. Add the dependency to your shard.yml:
    dependencies:
  xid:
    github: lost22git/xid.cr
    1. Run shards install

    Usage

    require "xid"

# generate a Xid
xid = Xid.generate

# xid to string
str = xid.to_s

# parse string to Xid
xid2 = Xid.from_s str

# xid as read only bytes view
bytes = xid2.as_bytes

# generate Xid by bytes
xid3 = Xid.from_bytes bytes

# resolve time from xid
p! xid3.time

# print xid debug info
xid3.debug
